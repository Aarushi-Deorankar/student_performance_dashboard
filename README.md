1. Project Title / Headline

Student Performance Analytics Dashboard
An interactive Power BI dashboard designed to analyze and visualize student academic performance across subjects, gender, and results, helping identify trends, strengths, and areas for improvement.

2. Short Description / Purpose

The Student Performance Dashboard is a beginner-level Power BI project that provides a clear and interactive overview of student marks, pass/fail distribution, and subject-wise performance.
It is designed to demonstrate data visualization, basic analytics, and dashboard design skills using Power BI.
